Installation
============

The NBG Python SDK can be installed from PyPI with Poetry__, Pipenv__ or pip__:

__ https://python-poetry.org
__ https://pipenv.kennethreitz.org
__ https://pip.pypa.io/en/stable/

.. code-block:: bash

    $ poetry add nbg  #, or
    $ pipenv install nbg #, or
    $ pip install nbg


Requirements
------------

- **Python**: 3.6, 3.7, 3.8
